## Cara Penggunaan

1. Clone repo:
   ```bash
   git clone https://github.com/kimberlytumober/Aplikasi-manajemen-kontak-sederhana.git
   ```
2. Masuk ke direktori:
   ```bash
   cd Aplikasi-manajemen-kontak-sederhana
   ```
3. Run
   ```bash
   python 'Aplikasi Manajemen Kontak Sederhana.py'.py
   ```
